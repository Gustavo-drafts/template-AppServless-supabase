# Getting Started

## In terminal run:

```bash
Load dependecies:

npm install
# or
yarn 

---------

start app:

npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


Routes from navegate in aplication:

`'/'` = Login screen

`'/Register'` = Register screen

`'/Dashboard'` = Capture screen
